# 여행모아 앱서비스 백엔드
---------------------------------------
- 해당 프로젝트는 대한민국 내에서 여행자들에게 장소 추천과 개인 맞춤형 여행 경로를 추천하고 공유하는 편리한 플랫폼을 개발하는 것에 목적을 두고 있습니다.

# 프로젝트 참여자
---------------------------------------
- 강태영
- 황기연
- 유범용

# Project
---------------------------------------
- OS : Windows10
- Language : Java
- Framework : Spring Boot, JPA, QueryDSL
- Tool : Intellij
- DB : MySQL
